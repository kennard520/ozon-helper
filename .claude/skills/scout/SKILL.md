# 跨境选品可行性分析 Skill

## 用途

输入一个产品关键词（中文或英文），自动完成：
1. **搜 1688 进价** — 找到供货价区间和重量
2. **搜 4 平台竞品价** — AliExpress / Temu / TikTok Shop / Ozon 上同类产品的在售价格
3. **调用定价器** — 用 calc.py 算出各平台的保本价和推荐售价
4. **对比出结论** — 竞品价 vs 我们的保本价，判断每个平台能不能做

## 调用格式

```
/scout 宠物牌 定制刻字 不锈钢
/scout 可降解宠物拾便袋
/scout silicone folding storage box
/scout LED 宠物项圈
```

也可以带已知参数缩短搜索：
```
/scout 宠物除毛手套 进价=8 重量=30
/scout 桌面收纳盒 进价=12 重量=50 类目=收纳整理
```

## 执行步骤

### Step 1: 搜 1688 进价

用 WebSearch 搜索：`1688 {产品关键词} 一件代发` 或 `site:1688.com {关键词}`

提取：
- **进价区间**（最低价 / 主流价 / 品质款价）
- **重量估算**（从商品描述或常识推断）
- **起订量**（是否支持一件代发）
- **产地**（义乌/深圳/东莞等）

如果用户已给 `进价=X 重量=Y`，跳过此步。

### Step 2: 搜 4 平台竞品价

依次搜索：

1. **AliExpress**: `aliexpress {英文关键词} price` 或 `site:aliexpress.com {英文关键词}`
2. **Temu**: `temu {英文关键词} price` 或 `site:temu.com {英文关键词}`
3. **TikTok Shop**: `tiktok shop {英文关键词} price` （数据可能有限）
4. **Ozon**: **优先用本仓自带 scraper**，比 WebSearch 精准 10 倍：
   ```bash
   $env:PYTHONPATH = "tools/ozon-scraper"
   python -m ozon_scraper scout "{俄语关键词}" --limit 30
   ```
   输出 JSON 含：
   - `price.min / p25 / median / p75 / max` —— 价格分布（RUB）
   - `discount_pct.median` —— 普遍折扣率
   - `review_count.median / max / gte_1000` —— 市场体量信号
   - `top_3_by_reviews` —— 头部 3 个 SKU（URL、价、评论数）

   主流价取 `price.p25 ~ price.p75` 区间；如果 `review_count.gte_1000` 占比高，说明这是红海类目，要算清楚利润空间。
   只在 scraper 失败或类目过冷门（结果 < 5）时回退到 WebSearch `ozon {俄语关键词} цена`。

提取每个平台的：
- **价格区间**（最低价 ~ 最高价）
- **主流价位**（销量最高的价格带，Ozon 用 p25-p75）
- **货币**（USD / RUB / THB 等）

### Step 3: 调用定价器

用 Step 1 的进价和重量，调用 calc.py：

```bash
python .claude/skills/price/calc.py --cost {进价} --weight {重量} --category {类目} --output json
```

如果用户没指定类目，根据产品自动判断最匹配的类目。

从 JSON 结果中提取每个平台的：
- `breakeven_cny`（保本价）
- `target_cny`（推荐售价）
- `target_profit_abs`（每单利润）
- `return_loss_per`（退1单亏多少）
- `orders_to_cover`（退1单要几单补）
- `inputs.return_rate`（退货率）

### Step 4: 对比 & 出结论

对每个平台，比较：
- **竞品价 vs 我们的保本价** → 能不能做
- **竞品价 vs 我们的推荐售价** → 利润空间大不大
- **退货抗性** → 退1单要几单补

## 输出格式（用 markdown 表格）

### 产品信息

> **{产品名}** · 1688 进价 ¥{X}-{Y}（取 ¥{Z}） · 约{W}g · 类目：{类目}
> 汇率实时 · 退货率按平台×类目数据

### 竞品价 vs 我们的成本

| 平台 | 竞品在售价 | 我们保本价 | 我们推荐价 | 差距 | 每单赚 | 退率 | 补几单 | 判定 |
|---|---|---:|---:|---|---:|---:|---:|---|
| AliExpress | $X-Y | ¥Z | ¥W | 说明 | ¥N | X% | N | ✅/⚠️/❌ |
| Temu | $X-Y | ¥Z | ¥W | 说明 | ¥N | X% | N | ✅/⚠️/❌ |
| TikTok SEA | ฿X-Y | ¥Z | ¥W | 说明 | ¥N | X% | N | ✅/⚠️/❌ |
| Ozon | ₽X-Y | ¥Z | ¥W | 说明 | ¥N | X% | N | ✅/⚠️/❌ |

### 判定规则

- **✅ 能做**：推荐价 ≤ 竞品主流价，有利润空间
- **⚠️ 勉强**：推荐价在竞品价上端，或保本价接近竞品价，利润薄
- **❌ 做不了**：保本价 ≥ 竞品价，数学上不可能盈利

### 结论

- **推荐平台**：哪些平台能做，优先级排序
- **放弃平台**：哪些做不了，原因（运费太高/竞品太卷/退货率太高）
- **风险提示**：定制品？标品红海？合规风险？季节性？
- **下一步**：具体的 1688 搜索词 + 建议操作

## 类目自动判断

根据产品关键词匹配类目：
- 宠物/猫/狗/鱼/鸟 → 宠物配件
- 收纳/整理/储物/折叠/箱/盒 → 收纳整理
- 文具/笔/本/办公/桌面 → 办公文具
- 装饰/挂件/摆件/花瓶/画 → 家居装饰
- 圣诞/万圣/复活节/节日/派对 → 节日礼品
- 定制/刻字/个性化/照片/纪念 → 定制类
- 口红/眼影/美甲/护肤 → 美妆
- 衣服/T恤/裙/裤/鞋 → 服装
- 以上都不匹配 → 通用

## 注意事项

- 竞品价格搜索结果可能不完整，标注数据来源和搜索时间
- Ozon 用俄语搜效果更好
- TikTok Shop 竞品价数据较难获取，可标注"数据有限"
- 如果竞品价差异很大，取销量最高的价格带作为主流价
- 对比时统一换算为 CNY，用 params.json 中的实时汇率
